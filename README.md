# java-cli

Simple Java command-line example (Maven)

Quick start

- Build:

```fish
cd java-cli
mvn package
```

- Run the packaged jar:

```fish
java -jar target/java-cli-0.1.0.jar arg1 arg2
```

- Or run with `mvn exec:java` (requires `exec` plugin configuration if needed).

Notes

- Requires Java 11+ and Maven installed.
